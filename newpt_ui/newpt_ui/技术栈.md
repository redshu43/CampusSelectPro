### 响应式前端
响应式：就像一个自动跟踪系统，当数据发生变化时，界面会自动更新，当用户在界面输入时，数据会自同步

#### 传统JS中的表现：
```javascript
let name = "lcy"
console.log(name) // "lcy"

name = "lcy"      // 数据变了，但页面不会自动更新
// 需要手动操作DOM：document.getElementById('name').textContent = name
```
#### VUE响应式：
```javascript
import { ref } from 'vue'

const name = ref("lcy")
// 当name.value = "lcy"时，所有使用name的地方都会自动更新
```
vue3中使用reactive响应式创建对象
js基本类型并非响应式，而用vue3的ref包装之后，可以创建响应式数据，让基本类型的数据变成响应式的
